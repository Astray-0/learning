# Project: Recipes

This project utilize the Basic HTML knowledge and put it into practice for building a basic recipe website.

This website consists of a main index page which will have links to a few different recipe.

## Knowledge check
This project demonstrates the learning of below tags.
- anchor element `<a>`
- image embed element `<img>`
- list element `<ul>`, `<ol>`, and `<li>`
- section heading element `<h1>`
